# WiredWorld
> This is a Django website project, assigned by SoftUni for an exam.

To run it locally you will need:
 - Postgres database
 - install all packages using ```pip install -r requirements.txt```
 - .env file that needs to have:
    - SECRET KEY
    - DEBUG (1 or 0)
    - ALLOWED_HOST
    - DB_NAME
    - DB_USER
    - DB_PASSWORD
    - DB_HOST
    - DB_PORT
